# Python Assignments

A collection of laboratory work and practical projects focused on intermediate Python development, software design patterns, and testing.

## Key Accomplishments

* **Math CLI Tool:** Developed a command-line application to calculate trigonometric, hyperbolic, and derivative functions with 3-decimal precision.
* **Advanced OOP & Design Patterns:** Implemented core OOP principles (inheritance, encapsulation, abstract classes) and decoupled object creation using an extendable **Factory Method pattern**.
* **Configuration Engineering:** Built robust configuration systems that parse CLI arguments (`argparse`), load YAML data, and map them onto typed `dataclasses`.
* **Defensive Coding & Custom Exceptions:** Created domain-specific exception hierarchies to validate user inputs, registration forms, and reservation systems.
* **Automated Testing (`pytest`):** Developed extensive test suites using parameterization, exception assertions, and temporary file directories (`tmp_path`) for testing CSV data lifecycles.
